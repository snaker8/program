IV. 학생이 지켜야 할 프로세스

1

수업 시작 즉시 — 리뷰 테스트

전 수업 내용 복습 테스트를 먼저 진행. 오답은 본인이 정리한 후, 자기 풀이 시간에 선생님께 개별 질문.

2

모든 빈 시간 — 즉시 오르조

개념 설명 차례를 기다리는 동안, 본 수업이 일찍 끝났을 때, 중간에 빈 시간이 생겼을 때 — 1분이라도 비면 즉시 오르조 학습교재로 넘어감. 휴대폰·잡담 X.

절대 하지 않는 것 NEVER

  • 리뷰 테스트 건너뛰기 — 매 수업 첫 10분은 무조건 리뷰
  • 개념 노트 미완성 상태로 연습문제 진행 — 미완료 시 반드시 재설명

  • 확인 테스트 결과를 다음 날 입력 — 반드시 당일
  • 학생 답을 직접 알려주기 — 개입은 "어디서 막혔어?" 까지만
  • 남는 시간에 휴대폰·잡담 허용 — 즉시 오르조로 넘어감
