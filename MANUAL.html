  1. /diagnostics/prep/{class_id} 열기

  5. 카드와 시트를 학생 자리에 배치

90분 5단계 진행 (시차 배치)

단계 ① — 리뷰 테스트 (수업 시작 즉시 10분)

모든 학생 동시 시작. 전 회차 학습 내용 복습 테스트.

  • 학생: 풀이 → 본인 채점 → 오답 정리
  • 담임: 시간 관리, 부정행위 감독
  • 오답은 본인이 메모해뒀다가 자기 풀이 시간에 개별 질문

이 단계가 빠지면 망각이 누적됩니다. 매 수업 무조건 첫 10분.

단계 ② — 개념 설명 (시차 배치 10~30분 구간)

5분씩 어긋나게 학생 한 명씩 1:1 개념 설명.

시점 학생 담임 행동
+10분 A에게 5~10분 1:1 개념 설명 + 개념노트 작성 안내
+15분 B에게 동일
+20분 C에게 동일
+25분 D에게 동일
+30분 E에게 동일

개념 설명 룰: - 학생 시트의 "미니강의 키 포인트" 그대로 시작 - 설명 후 학생이 직접 서술형 개념노트를 작성하게 함 - 끝나면 "다 풀고 막히면 손 들어"

단계 ③ — 자기 풀이 (개별 진입, 평균 30~70분)

학생마다 진입 시점이 다름. 두 단계로:

① 개념 확인 학습 (개념노트가 정확한지)
        ↓
   미완료 시 다음 단계로 못 감 → ②번 단계로 돌려보내기
        ↓
② 확인 연습문제 15~20문제
        ↓
   채점 + 1:1 피드백

담임 행동: - 호출 들어온 순서대로 응대 - 한 학생당 3분 이상 안 씀. 더 길어지면 "잠깐 다음 단계 가보고 안 되면 다시 부르자" - 호출 없으면 5분마다 한 바퀴 순회. 어깨 너머로 풀이만 확인 - 개입 금지: "어디서 막혔어?"만 물어보기. 답을 직접 알려주지 않음

단계 ④ — 확인 테스트 (개별 진입, 학생 시작+70분 시점)

학생 시계로 70분 시점에 시작. 7~10문항, 10분 제한.

  • 시트는 뒤집어 두고 신호와 함께 시작
  • 종료 후 즉시 채점
  • α/β/γ 등급 매김
  • 1:1 피드백 5분 (오답 1~2문항 짚기)

단계 ⑤ — 오르조 (모든 빈 시간)

오르조는 본 수업 종료 후 잔여 시간뿐 아니라 수업 중 발생하는 모든 빈 틈을 메우는 도구입니다.

빈 틈 발생 시점 누가 대처
개념 설명 차례 대기 (학생 B~E) 5~20분 빈 시간 즉시 오르조
자기 풀이 일찍 끝남 임의 학생 즉시 오르조
본 수업 종료 후 (학생 A 등) 잔여 5~10분 즉시 오르조

룰: - 휴대폰·잡담 허용 X - 1분 이상 비면 즉시 오르조로 전환 - 소단원 학습 교재 진행 (난이도 매트릭스 참고) - 진행 분량은 다음 수업 시작 시 점검

수업 후 5분

  1. 카드의 "수업 후 메모"란 채우기
  2. /diagnostics/entry에서 확인 테스트 결과 입력 (5명, 5분 안에)
  3. 결석 학생 표시 (보강 일정 잡기)

호출 시간 길어질 때 대처

상황 대응
개념 자체를 모름 개념노트로 돌아가 재설명. 다음 회차에 다시 미니강의로 다루자 메모
한 문항 5분 이상 막힘 같은 단원 더 쉬운 문항으로 우회
계산 실수 반복 시트 한 페이지 단위 계산만 풀게 → 채점 시 원인 태그 "계산"
학생이 다 풀어버림 즉시 오르조 학습교재로 전환

절대 하지 않는 것

  • 리뷰 테스트 건너뛰기 — 첫 10분은 무조건 리뷰
  • 개념노트 미완성 상태로 연습문제 진행
  • 개념 설명을 15분 이상 늘리기
  • 한 학생에게 한 회차 3개 이상 단원
  • 확인 테스트 결과를 다음 날 입력
  • 학생 답을 직접 알려주기
  • 남는 시간에 휴대폰·잡담 허용

케어 합계 5.0 초과 시

반 등록 화면의 care_weight 합계가 5.0 넘으면 ● 과부하 표시.

  • 가장 자기주도 학생을 다른 시간대로 이동 (가능하면)
  • 또는 그 반 정원을 4명으로 일시 축소
  • 매월 1일 단계 재평가 회의에서 조정

단계 재평가 기준 (매월 1일 또는 학기 전환 시)

  • α 단원이 해당 학기 단원의 80% 이상 → 다음 단계로 승격 검토
  • γ 단원이 3개 이상 누적 → 한 단계 내려서 복구 트랙
  • 망각 체크에서 이전 단원 70% 이하 → 단계 유지하되 복습 비중 증가
  • 교재 난이도(L1~L6) 매칭 재조정
